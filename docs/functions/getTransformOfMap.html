Function getTransformOfMap

  • getTransformOfMap(): TransformOfMap

  • 获取所在位置的所在地图区域并重映射坐标区域。

    参数
    • x 获取到的人物的x轴坐标位置。
    • y 获取到的人物的y轴坐标位置。
    • mapId 获取到的人物的所在地图区域。

    返回值
    • true 表示坐标位置获取成功。
    • false 表示坐标位置获取失败，此时三个参数均不会被改变。

    说明

    地图区域的值的解释

    地图编号 含义
    0 提瓦特大陆。
    1 渊下宫
    2 地下层岩

    无论成功与否都会设置 LastErr 值。调用 GetLastErr() 可以获取错误码。

    调用本函数前，如果没有初始化过，或最后一次初始化返回失败，则需要调用 init() 函数进行初始化。

    Returns

    获取到的人物的所在地图区域。
